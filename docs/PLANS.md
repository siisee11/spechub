# Execution Plans

Complex work in this repository is tracked as checked-in execution plans.

## Location

- Active work: `docs/exec-plans/active/`
- Completed work: `docs/exec-plans/completed/`
- Ongoing debt and follow-ups: `docs/exec-plans/tech-debt-tracker.md`

## Plan Template

```md
# <task title>

## Goal / scope

## Background

## Milestones
- [ ] milestone 1
- [ ] milestone 2

## Current progress

## Key decisions

## Remaining issues / open questions

## Links to related documents
```

## Rules

- Each milestone should fit in a single focused coding iteration.
- Update the plan as the source of truth while work is in flight.
- Move the file to `completed/` when the work lands.

# Create Harness

## Goal

Turn `create-harness/` into a working, testable, self-hosted example of the harness engineering system it describes.

## Users

- repository maintainers who want to bootstrap a new project with a portable harness
- coding agents that need deterministic docs, commands, and runtime contracts

## Core Outcomes

- clear repository map
- deterministic worktree boot metadata
- local observability query surfaces
- recurring cleanup scans and grading
- automated agent loop scaffolding for prompt-to-PR workflows
